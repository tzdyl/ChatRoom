# 完整的项目在develop分支上，develop分支上的内容以master分支上内容为基础，稍有调整，实现了网络聊天室的基本功能
 
## 网页聊天室功能如下：
* 登录
* 注册
* 聊天，显示在线人数，消息群发，私聊等功能
* 查看登录人员信息
* 修改登录人员基本信息，修改密码
* 页面注销
* 查看log日志

## 使用技术
* 前端页面

  使用angular框架结合bootstrap实现页面的展示

* 后端实现

  使用springmvc+mybatis+mysql实现
  
 `该项目实现了前后端分离`
 
 ## 数据库
 主要包含两个表user表和log表
 * user----用来保存用户的基本信息
 * log----用来保存用登录以及更新信息的日志
